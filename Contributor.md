## How to Contribute

### 1. Fork the repo on GitHub
Click the **Fork** button on the GitHub repository page.

### 2. Clone your fork
```bash
git clone https://github.com/their-username/testdrive.git
```

### 3. Create a feature branch from roboapps
```bash
git checkout -b add-myapp origin/roboapps
```

### 4. Add your app inside the `roboapps/` folder
```
e.g. roboapps/MyApp/myapp.py
```

### 5. Push to your fork
```bash
git push origin add-myapp
```

### 6. Open a Pull Request on GitHub
- **From:** `their-fork:add-myapp`
- **To:** `Roboworks-Global/testdrive:roboapps`
